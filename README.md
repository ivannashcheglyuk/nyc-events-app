# WEB103 Project 1 - *NYC Events*

Submitted by: **Ivanna Shcheglyuk**

About this web app: **The NYC Events App is a web application that displays upcoming events in New York City. Users can browse events on the homepage presented as attractive cards with images, titles, and key details. By clicking on a card, users can view a detailed page for each event, including a short description. The app is built using only HTML, CSS, and JavaScript, styled with PicoCSS, and designed to be responsive and visually appealing.**

Time spent: **9** hours
